Light Run

Status: HIATUS

Engine: Unity

Language: C#

Tools: Unity, Visual Studio Code

Afraid of the dark was a prototype I created to better understand Unity's built in lighting functionality.

The core idea is the player is afraid of the dark and loses health when he is in the dark, to replenish health a player must stand in a light. The player will complete some mini challenges on the way to the end of the level.

The player has the ability to run, pickup, drop and throw items. Using these abilities the player must complete a challenge between each light.

First challenge : A short zigzag corridor that the player has to sprint through.

Second challenge : Using the blocks under the light the player has to get over the wall.

Third challenge: Time a run past a rotating spotlight, if seen then its game over.
