Reverse Sim

Status: In development

Engine: Unity

Language: C#

Tools: Unity, Visual Studio Code

Code Snippits: CarGameplay, GameControl

The idea behind this game came after a silly discussion I had over lunch with some friends about reversing a car. I decided I could make a game out of it. I decided to make it for mobile as I wanted to learn more about mobile game development.

I wanted to make the game feel unique to mobile. To accomplish this I used the core functionality found in current mobiles. Using the accelerometer to turn, the player has to reverse a car into parking space within a set time limit. I have also implemented the swipe function for changing view, for example swiping left switches the camera to look over the left shoulder.
